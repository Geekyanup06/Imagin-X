# 🎨 ImaginX – AI Text to Image Generator

ImaginX is an AI-powered **Text-to-Image Generator** that allows users to create stunning, high-quality images simply by entering a text prompt.  
The project is powered by the **Hugging Face API** and offers a sleek, futuristic user interface with both light/dark theme modes.

## ✨ Features
- 🖌️ **Text-to-Image AI** – Generate images instantly from your custom prompts.
- 🌗 **Light & Dark Theme Modes** – Switch between bright and dark UI themes.
- 🧠 **Auto Caption Generator** – Get interesting pre-built prompts to inspire creativity.
- 🎨 **Attractive UI** – Modern, minimalistic, and responsive design.
- ⚡ **Fast & Smooth Performance** – Powered by Hugging Face API for quick image generation.

---

## 🛠️ Tech Stack
- **Frontend:** HTML, CSS, JavaScript
- **Backend/API:** Hugging Face API (Text-to-Image model)
- **Design:** Responsive & futuristic UI/UX

## 🚀 Getting Started

### 1️⃣ Clone of the Repository
https://github.com/Geekyanup06/Imagin-X




2️⃣ Open the Project
Simply open the index.html file in your browser.

⚠️ Note: You need a Hugging Face API key to run the project.

🔑 Hugging Face API Setup
Create an account on Hugging Face.

Go to Settings → Access Tokens and create a new API key.


Replace the placeholder API key in the JavaScript code:
const API_KEY = "YOUR_HUGGING_FACE_API_KEY";
